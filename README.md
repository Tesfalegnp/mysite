## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Git

### 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Tesfalegnp/mysite.git
   cd mysite 
2. **Install dependencies**

```bash
npm install
# or
yarn install
```
Run development server

```bash
npm run dev
# or
yarn dev
```
Open your browser
Navigate to http://localhost:5173

🏗️ Build for Production
bash
npm run build
# or
yarn build
The build output will be in the dist folder.

### 🔄 CI/CD Pipeline
